Welcome to Haiphen's Documentation Page!
===================================

**Haiphen** (/hi'fen/) is a security protocol for protecting enterprise infrastructure. 
Our _`digital copyright <https://github.com/haiphenai/digital-copyright>`_  uses 'zero-trust' ground-up integration to verify the identity of each node involved in any data transfer. It's lightweight (60kb) and makes no external API calls. We mount directly to your existing servers and plugin directly to your CICD integrations. 

We close the loop information security by simulating blue team attacks on our infrastructure leveraging MITRE Caldera
